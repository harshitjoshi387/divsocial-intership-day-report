# 🎨 Day 4 – CSS Fundamentals & Mini Projects

> Learning CSS fundamentals by styling a resume and building small UI components.

---

# 🎯 Objective

The objective of this task was to understand the basics of CSS and apply them by styling an HTML resume and creating reusable UI components.

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- Google Chrome

---

# 📚 Topics Covered

- CSS Boilerplate
- External CSS
- CSS Selectors
- Colors
- Fonts
- Margin & Padding
- Border
- Border Radius
- Box Shadow
- Hover Effects
- Flexbox Basics

- Difference between Inline, Internal and External CSS.
- How CSS Selectors target HTML elements.
- Importance of separation of structure (HTML) and styling (CSS).
- How margin and padding affect layout.
- How to use Flexbox for alignment.
- How hover effects improve user experience.
- How to organize CSS into reusable components.
