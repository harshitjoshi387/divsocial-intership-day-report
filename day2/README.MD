---

# 🌐 DNS Lookup

## Command Used

```bash
nslookup github.com
```

## Output

```text
Server: reliance.reliance
Address: 2405:201:6844:8be::c0a8:1d01

Non-authoritative answer:

Name: github.com
Address: 20.207.73.82
```

### Explanation

The Domain Name System (DNS) converts a human-readable domain name into an IP address.

When the browser receives the IP address of **github.com**, it establishes a secure connection with the GitHub server and begins downloading the webpage resources.

---

# 🌍 Browser Loading Flow

```text
User enters github.com
        │
        ▼
DNS Lookup
        │
        ▼
IP Address Found
        │
        ▼
TCP Connection
        │
        ▼
TLS Handshake (HTTPS)
        │
        ▼
HTTP Request
        │
        ▼
Server Response
        │
        ▼
Browser Downloads Resources
        │
        ▼
Page Rendered
```
